—¿Cómo es que se instalan paquetes con yarn?
Para instalar paquetes con Yarn, puedes usar el comando yarn add nombre-del-paquete. Este comando descargará el paquete y lo agregará al archivo package.json.
—¿Cómo es que se borran paquetes con yarn?
Para eliminar paquetes con Yarn, debes usar el comando yarn remove nombre-del-paquete. Este comando eliminará el paquete de tus dependencias directas y actualizará tus archivos package.json y yarn.lock en el proceso.
—¿Hay algún equivalente en yarn que haga lo mismo que npm init?
El equivalente de npm init en Yarn es yarn init. Este comando crea un nuevo archivo package.json.
